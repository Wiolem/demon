# demon

FE-Map-Enum vscode's Extension.

## Features

```
WAITING(10, "待处理"),
PREPROCCESSED(20, "预处理完成"),
ORDER_CREATED(30, "单据已创建"),
FINISHED(40, "已完成"),
```

> cmd + shift + p
> MapEnum

```
{
  FINISHED: 40,
  ORDER_CREATED: 30,
  PREPROCCESSED: 20,
  WAITING: 10
  properties: {
    10: { name: "待处理", value: 10 },
    20: { name: "预处理完成", value: 20 },
    30: { name: "单据已创建", value: 30 },
    40: { name: "已完成", value: 40 }
  }
}
```

## Requirements

install vscode.

## Extension Settings

nothing.
